How to run many version of MStorage

Sometimes you will need to have a couple or three storages on your hard drive and in the same time have many running application of MStorage.

MStorage save some settings for your convenience. It save all yours settings in your home directory in .MStorage/settings. If all running version of MStorage will use it, there will be flotsam and jetsam of settings.

For each application you can set name of settings file. To end of your running command add option: 

--settings=name_of_settings_file

Now all settings of this version of application will save in ~/.MStorage/name_of_settings_file.
