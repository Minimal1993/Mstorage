Additional features

The first advantage is MStorage save it tree in simply type - as text files and images. If you, for example, create MStorage archive in Dropbox or Google Drive directory, you will have access for MStorage tree from any your device and any place you want.

MStorage has the same menus in menu "File". But you have to have understanding, menu in "File" is menu for opened document in current active tab. If you choose menu in tree, menu called for current selected item in tree.

You can move file and directory into MStorage tree. File will moved whith his pictures and directory will moved with all content into it.

If you choose "Save this file as..." you can select any directory on your hard drive, and file will copied into it.

It can be useful to hide toolbar and tree. For this check out View -> "View menu toolbar" and View -> "View storage tree model".
