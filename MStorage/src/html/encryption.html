Please, note! AES 128-bit encryption released in MStorage is not guarantee absolutely security of your data. If you want more stronger encryption, please look for especial applications for this purposes. Encryption in MStorage is intended to make base security. Because of open source code of this project and developer's weakness in encryption theme, you mustn't encrypt important documents by MStorage.
