Additional features

The first advantage of MStorage is save storage tree in simple and common types, - directories, text files and images. If you, for example, will create MStorage archive in Dropbox or Google Drive directory, you will have access for MStorage tree from any of your device and any place what you want.

You can move file and directory into MStorage tree. File will moved with all his pictures; directory will moved with all content into it.

If you choose "Save this file as..." you can select any directory on your hard drive, and file will copied into it.

It can be useful to hide toolbar and tree. For this you have to switch on View -> "View menu toolbar" and View -> "View storage tree panel"(Or type Ctrl+T).
